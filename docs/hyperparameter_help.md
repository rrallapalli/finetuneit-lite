# Hyperparameter Help Text

The Streamlit UI includes short help pop-ups for key fields.

## Covered fields

- Dataset sample size
- Validation split
- Prompt template
- Base model
- Max sequence length
- 4-bit loading
- Batch size
- Gradient accumulation
- Max steps
- Learning rate
- Warmup steps
- Weight decay
- Logging steps
- Eval steps
- Save steps
- Optimizer
- LR scheduler
- Seed
- LoRA rank
- LoRA alpha
- LoRA dropout
- rsLoRA
- Output directory
- Hugging Face adapter repo

These are intentionally brief so non-expert users can configure experiments without reading model training documentation first.
